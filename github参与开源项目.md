### 8.想给这个项目加一些功能

​	1.首先创建一个分支

```
git branch demo2.0
```

​	2.切换分支，在demo2.0这个分支进行创作不会影响master分支

```
git checkout demo2.0
```

​	3.熟练后可以合并一起写

>git checkout -b demo2.0
